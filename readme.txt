== Description ==

with this plugin, you can easily add paystar.ir gateway to learnpress lms
با این پلاگین میتونید درگاه پرداخت پی استار رو برای پرداخت هزینه دروس و دوره های افزونه لرن پرس اضافه کنید


== Installation ==
 
just enter paystar.ir terminal (API-Key)
راه اندازی و پیکربندی افزونه نیاز به تنظیمات خاصی نیست. فقط کافیه کد ترمینال پذیرندگی خودتون رو در تنظیمات افزونه وارد کنید
برای ورود به صفحه تنظیمات افزونه از منوی لرن پرس ، زیرمنوی تنظیمات ، به سربرگ درگاه های پرداخت رفته و روی اسم درگاه پی استار کلیک کنید
